# D2C Fashion Brand – Customer Retention & Loyalty Strategy

## Project Overview
This project analyzes customer transaction data from a Direct-to-Consumer (D2C) fashion brand to evaluate customer loyalty, promotional effectiveness, and retention opportunities. Using Python, SQL, and Power BI, the project identifies high-value customer segments, measures discount dependency, and develops actionable strategies to improve profitability and customer lifetime value.

## Objectives
* Identify loyal vs. discount-dependent customers.
* Evaluate the effectiveness of promotional campaigns.
* Develop a customer retention framework.
* Define an Ideal Customer Profile (ICP).
* Provide data-driven recommendations to improve long-term profitability.

## Tools & Technologies
* Python (Pandas, NumPy)
* SQL
* Power BI
* Git & GitHub

## Key Analysis
### Customer Segmentation
Customers were segmented based on:
* Revenue Contribution
* Average Order Value (AOV)
* Promotional Dependency
* Customer Satisfaction Ratings

### Promotional Effectiveness
Analysis showed that highly discount-dependent customers do not spend significantly more than full-price customers within the same revenue segment, indicating opportunities to reduce unnecessary promotional spending.

### Retention Strategy
Designed a data-driven retention playbook that includes:
* Promotional Sunset Strategy
* Loyalty-Based Reward Programs
* Customer Retention Initiatives
* Performance Monitoring Framework

### Ideal Customer Profile (ICP)
The highest-value customers were identified as:
* High-spending customers (~$91 AOV)
* Non-discount users
* Highly satisfied customers
* Premium-quality focused buyers

## Key Recommendations
* Reduce discounts for low-value, highly price-sensitive customers.
* Replace discounts with loyalty rewards for high-value customers.
* Focus marketing efforts on acquiring customers similar to the identified ICP.
* Prioritize retention initiatives to increase repeat purchases and customer lifetime value.

## Business Impact
This project demonstrates how customer analytics can be used to:
* Improve profit margins
* Reduce discount dependency
* Increase customer lifetime value (CLV)
* Strengthen customer retention strategies
* Support sustainable business growth through data-driven decision making
